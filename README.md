# VrSafe

## Installation

1. Installer [Tampermonkey](https://www.tampermonkey.net/) (Chrome / Firefox / Edge)
2. Cliquer sur [**vrsafe.user.js**](https://raw.githubusercontent.com/neufcentonze/VrSafe.Grepolis/main/vrsafe.user.js)
3. Tampermonkey détecte le userscript et propose l'installation — confirmer
4. Recharger la page Grepolis
